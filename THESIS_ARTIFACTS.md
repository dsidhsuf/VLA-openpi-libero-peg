# Thesis Reproduction Artifacts

This directory packages lightweight artifacts associated with the thesis experiments. Model checkpoints, generated datasets, videos, virtual environments, caches and bulk logs are intentionally excluded.

## Chapter 4: Waypoint Validation

- `thesis_artifacts/chapter4_waypoint_validation/trajectory_1/`: artifacts matching Table 4.5 (`obj_slot_xy = 0.015642 m`, `obj_slot_z = -0.003593 m`).
- `thesis_artifacts/chapter4_waypoint_validation/trajectory_2/`: artifacts matching Table 4.6 (`obj_slot_xy = 0.012838 m`, `obj_slot_z = -0.019278 m`).

Each trajectory contains a JSON summary and CSV action/state traces. Video files referenced by local experiment outputs are not included.

## Chapter 5: Fine-Tuning Diagnostics

- `thesis_artifacts/chapter5_training_frame_diagnostics/training_frame_replay_full_new_prompt_2000.csv`: matches Section 5.5.3 (`MAE_xyz = 0.02317`, finite-value `cos_xyz = 0.18904`, `z_sign = 0.6000`, `gripper_sign = 0.3833`).
- `thesis_artifacts/chapter5_training_frame_diagnostics/training_frame_replay_pi05_lora_v3_safe_3000_on_aug.csv`: diagnosis on augmented frames.
- `thesis_artifacts/chapter5_training_frame_diagnostics/training_frame_replay_pi05_lora_v3_safe_3000_on_original.csv`: comparison diagnosis on original frames.
- `thesis_artifacts/chapter5_training_frame_diagnostics/training_frame_replay_pi05_lora_v4_rare_only_1000_on_v4.csv`: rare-key-sample weighting diagnosis.
- `thesis_artifacts/chapter5_segment_augmentation/`: Section 5.5.5 closed-loop evaluation artifacts. The state trace starts with `eef_obj_dist = 0.27487 m` and records the segment-augmentation evaluation trajectory, ending at approximately `0.17 m`.

## Legacy Assets And Baselines

- `thesis_artifacts/legacy_benchmark_assets/init_states/`: early benchmark initial-state arrays generated by `code/generate_earbud_init_states.py`.
- `code/legacy/backup_proto_v1/`: early prototype object definition and scripted-pose test.
- `code/legacy/frozen_baselines/`: frozen controller baselines retained for experiment history.

## Excluded

- Model weights and checkpoints: `*.safetensors`, `*.bin`, `*.pt`, `*.pth`, checkpoint directories.
- Videos: `*.mp4`, `*.avi`, `*.mov`, `*.mkv`.
- Generated LeRobot datasets, raw episode image frames, caches and virtual environments.
- Official-suite smoke-test outputs that do not match the thesis Table 3.1 aggregate results.
